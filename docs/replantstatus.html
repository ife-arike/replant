Important Technical Notes

Supabase Project Live at supabase.com — org: ife-arike's Org, project: replant (Production). Migration v1.6.0 applied 29 Apr 2026. pgcrypto v1.3 confirmed live. API keys saved securely — anon key (client-safe), service_role key (server only, never in GitHub).
Heartcry Encryption pgp_sym_encrypt / pgp_sym_decrypt via pgcrypto. Base64 pattern required (SEC ruling). Key stored in Supabase Vault as 'heartcry_encryption_key'. Encryption in edge function before INSERT. Decryption in admin edge function only — decrypt_heartcry_content() EXECUTE restricted to service_role.
churches_public View All app-layer church queries must use the churches_public view — not the raw churches table. View has security_invoker = true. Strips region_admin_only column. Filters is_active = true automatically. Raw table access: admin dashboard / service_role only.
Logo SVG rp_vector.svg — viewBox="482 201 581 544" (website), viewBox="490 200 580 560" (card). Has 2 paths inside the symbol — both must be included for the full logo to render correctly.
Netlify Forms Form names are join-network and serve-with-us — must match exactly. Use URLSearchParams encoding for fetch POST — confirmed working.
PDF / Print HTML files cannot be converted to PDF reliably via wkhtmltopdf (no Google Fonts access, no CDN JS). Always use PNG export via wkhtmltoimage at 1050×600 for 300 DPI business card output.
Email Zoho alias: "support@" blocked by AS101 error — using connect@projectreplant.org instead. Resend API key stored in Supabase Vault as 'resend_api_key'.
Canva Connector Limits Editing operations available: replace_text, find_and_replace_text, format_text, insert_fill, delete_element, position_element, resize_element. No insert_text — new text boxes cannot be added programmatically. Canva may return cached design state when inspecting via get-design-pages.
GitHub Repo: github.com/ife-arike/replant (public). Auth via personal access token embedded in remote URL. Push to main branch directly. Dev roles doc at docs/replant-dev-roles.html.
